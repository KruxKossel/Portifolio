## 🚀 Sobre o Projeto

Portfólio pessoal desenvolvido com React e Tailwind CSS, apresentando projetos e habilidades de forma interativa. O site conta com animações suaves, design responsivo e interface moderna.

## 🛠 Tecnologias Utilizadas

- **Frontend:**
  - React 18.2.0
  - Tailwind CSS
  - React Router Dom
  - React Icons
  - GSAP (Animações)

- **Ferramentas de Desenvolvimento:**
  - Vite
  - ESLint
  - Prettier
  - PostCSS

## ⚙️ Funcionalidades

- **Home:**
  - Apresentação pessoal
  - Animação de chuva interativa
  - Links para redes sociais

- **Sobre:**
  - Biografia detalhada
  - Habilidades técnicas
  - Experiência profissional
  - Animação do corvo

- **Projetos:**
  - Filtro por tipo (Autoral/Colaborativo)
  - Cards interativos
  - Modal com detalhes
  - Links para repositórios e demos

## 🎨 Layout e Design

- **Tema:**
  - Cores principais: Roxo (#8B00FF) e Rosa (#DA70D6)
  - Efeitos de glass morphism
  - Animações suaves
  - Design responsivo

- **Componentes:**
  - Cards com hover effects
  - Modais com backdrop blur
  - Ícones animados
  - Navegação intuitiva

## 🚦 Como Usar Este Projeto

Este projeto está disponível como referência para a comunidade. Se você quiser usá-lo como base para seu próprio portfólio, siga estes passos:

1. **Faça um Fork:**
   - Clique no botão "Fork" no topo desta página
   - Isso criará uma cópia do projeto na sua conta do GitHub

2. **Clone seu Fork:**
   ```bash
   git clone https://github.com/SEU_USERNAME/Portifolio.git
   cd Portifolio
   ```

3. **Personalize:**
   - Modifique o conteúdo em `src/config/index.js`
   - Atualize as imagens em `src/assets/`
   - Adapte os componentes às suas necessidades
   - Altere as cores e estilos em `src/constants/theme.js`

4. **Instale e Execute:**
   ```bash
   npm install
   npm run dev
   ```

> 💡 **Dica**: Mantenha os créditos originais no rodapé ou em algum lugar do seu site. Não é obrigatório, mas é uma prática legal de reconhecimento!

## 📁 Estrutura do Projeto

```
src/
├── assets/         # Imagens e recursos
├── components/     # Componentes reutilizáveis
├── config/         # Configurações do projeto
├── constants/      # Constantes e temas
├── pages/          # Páginas principais
└── styles/         # Estilos globais
```

### Componentes Principais

- `Layout.jsx`: Container principal com navegação
- `ProjectCard.jsx`: Card de projetos reutilizável
- `ProjectModal.jsx`: Modal de detalhes do projeto
- `SocialLinks.jsx`: Links para redes sociais
- `RainEffect.jsx`: Animação de chuva interativa
